# reading-tree-project

읽기 트리(Reading Tree) 프로젝트 초기화 저장소.

## 다음 단계
1) Node 버전 확인 (LTS 권장)
2) 프레임워크 선택 후 스캐폴딩
   - Next.js: npx create-next-app@latest . --ts --app --tailwind --eslint --use-npm
   - Vite(React): npm create vite@latest . -- --template react-ts
3) 의존성 설치: npm install
4) 개발 서버 실행: npm run dev
